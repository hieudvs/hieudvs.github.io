Công việc mong muốn: Lập trình viên PHP

Vị trí: Nhân viên

Tôi rất thích ngành công nghệ phần mềm, tôi đã và đang tiếp tục học hỏi, thực thành thêm nhiều ngôn ngữ lập trình mới để thỏa niềm đam mê ...

Kỹ năng

Frontend
  • Html5 - Css3 - Bootstrap: Nắm vững cấu trúc, cách sử dụng ngôn ngữ dựng khung web, áp dụng vào dự án thực tế.
  • JavaScript - Jquery (Ajax): Kết hợp sử dụng các thư viện để tạo hiệu ứng cho web như: slide, truyền tải dữ liệu thông qua Ajax
  • Vuejs: sử dụng ở mức cơ bản.
Backend
  • Php: hiện tại đã và đang làm về CMS Wordpress cả cơ bản và nâng cao, đã làm qua các dự án thực tế về Laravel Framework.
  • CSDL:MySQL,nắm cách tạo, truy vấn và tương tác với database
Kỹ năng với cms Wordpress
  • Cắt giao diện từ file thiết kế (Photoshop - PSD, Allustrator – AI, hoặc các công cụ thiết kế online …), cắt lại giao diện theo mẫu website cho trước.
  • Viết theme Wordpress mới ( dùng starter theme tại https://underscores.me/), viết plugin
  • Sử dụng tốt các theme Wordpress có sẵn để đẩy nhanh tiến độ làm web, như theme Flatsome…
  • Có khả năng bóc tách code từ themes gốc để tạo phần quản lý admin dễ sử dụng cho người dùng ( Sử dụng plugin Advand custom fields ACF để tạo phần nhập liệu người dùng).
  • Tạo được shortcode, để linh hoạt cho việc sử dụng khi code
  • Tạo được các Post type/Taxonomy mới, tạo mới hoặc chỉnh sửa template để tối ưu giao diện cho từng post type.
  • Phân chia các file chức năng (các class), để dễ quản lý code và bảo trì, nâng cấp website. Sử dụng các hàm của Wordpress để thực hiện code (Như các hàm để lấy dữ liệu từ database WP_Query …)
  • Sử dụng tốt các plugin: Advand custom fields ACF, WP Crontrol (tạo các sự kiện chạy theo định kỳ), Polylang (tạo website đa ngôn ngữ), …
Công cụ lập trình
  • Visual Studio Code, PhpStorm: Hiện đang sử dụng PhpStorm thành thạo, áp dụng các plugin để giúp việc viết code nhanh hơn
  • Postman: Dùng được công cụ để kiểm tra các API
  • Git - Github - Bitbucker - Soucetree: Sử dụng được các công cụ quản lý code online, nắm các câu lệnh của git, hiện sử dụng Sourcetree - Bitbucker.
  • Telegram - Trello: dùng được các công cụ quản lý công việc ...
